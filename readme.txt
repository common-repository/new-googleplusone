== Description ==

Inserts Google + 1 Button in your site from the Dashboard. No code necessary!

You do not need to edit the theme. New GooglePlusOne is the easy way to insert the Google+1 in your site without have programming knowledges.


Features

Google + 1 will be added automatically in all your posts and pages. You can customize the place where the button will appear, by changing its position or by choosing another HTML element which will contain it.

Also, you can choose to have a single Google + 1 button in your site's header.

== Installation ==

1. Upload `new-googleplusone` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Read the Google + 1 page in Settings menu in wp-admin

== Frequently Asked Questions ==

How does this plugin work?

Very easy.

This plugin uses the Wordpress` hooks to modify your theme. It is not dangerous for your theme. This plugin does not affect your post content or layout. It only inserts Google + 1 before or after your posts.

Does this plugin works on all WP Themes?

Yes, we tested it on Twentyeleven Theme and on other free themes. 
Notice that in some cases the desired layout can be appear missaligned.
For advanced users: If you want to use your custom WordPress theme, you must use <?php post_class(); ?> in your template. For more details, see <a href="http://codex.wordpress.org/Function_Reference/post_class">http://codex.wordpress.org/Function_Reference/post_class</a>
